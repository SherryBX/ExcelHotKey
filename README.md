Excel 快捷键增强工具，使用 AutoHotkey 实现。

## 功能

### 1. 快速切换工作表
- **Ctrl + ]** - 切换到下一个工作表（映射为 Ctrl+PageDown）
- **Ctrl + [** - 切换到上一个工作表（映射为 Ctrl+PageUp）

### 2. 选中当前行
- **Ctrl + Shift + H** - 快速选中当前单元格所在的整行
  - 自动定位到行号并点击
  - 不受合并单元格影响

### 3. 合并单元格并居中
- **Alt + C** - 合并选中的单元格并居中对齐

## 使用方法

1. 安装 [AutoHotkey](https://www.autohotkey.com/)
2. 下载 ExcelHotKey++.ahk 文件
3. 双击运行脚本
4. 在 Excel 中使用上述快捷键

## 注意事项

- 脚本仅在 Excel 中生效
- 需要 Excel 处于活动窗口状态
- 建议将脚本添加到开机启动项
